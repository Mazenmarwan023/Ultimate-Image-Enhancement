# Ultimate-Image-Enhancement

## üìå Overview

This project is a comprehensive implementation of fundamental **Image Processing** techniques, with all core algorithms‚Äîincluding filtering, convolution, and edge detection‚Äîcoded **from scratch** using Python and NumPy. It covers noise manipulation, low-pass and high-pass filtering, advanced edge detection (like Canny), histogram analysis, and the creation of **Hybrid Images**.

---

## üöÄ Features

- ‚úÖ **Noise Simulation:** Gaussian, Uniform, and Salt & Pepper noise.
- ‚úÖ **Custom Filtering:** Average, Gaussian, and Median filters implemented via manual 2D **convolution**.
- ‚úÖ **Advanced Edge Detection:** Sobel, Prewitt, Roberts, and a **from-scratch Canny Edge Detector**.
- ‚úÖ **Contrast Enhancement:** Histogram Equalization and Image Normalization.
- ‚úÖ **Adaptive Thresholding:** Global and Local (Adaptive) thresholding for segmentation.
- ‚úÖ **Frequency Domain Filtering:** Low-Pass (LPF) and High-Pass (HPF) filtering using **Fourier Transform**.
- ‚úÖ **Hybrid Image Generation:** Combining LPF and HPF images for multi-scale perception.

---

## üß† Methodology (Core Implementation)

### 1. Noise and Filtering
All filters utilize a **custom 2D convolution function** to apply kernels manually.
* **Noise:** Functions for **Gaussian**, **Uniform**, and **Salt & Pepper** noise addition, applied to both grayscale and color images.
* **Low-Pass Filters:** Implementation of the **Average**, **Gaussian** ($\sigma$ controlled), and **Median** filters to demonstrate noise reduction.

### 2. Edge Detection
* **Gradient-Based Masks:** Implemented **Sobel**, **Prewitt**, and **Roberts Cross** operators to compute intensity gradients (edges).
* **Canny Detector:** A complete, custom implementation including **Gaussian smoothing**, **non-maximum suppression**, and **hysteresis thresholding**.



### 3. Histogram and Contrast
* **Histogram Analysis:** Functions to calculate and plot the **Histogram** and **Cumulative Distribution Function (CDF)**.
* **Equalization:** Implementation of **Histogram Equalization** to enhance image contrast by redistributing pixel intensities based on the CDF.

### 4. Frequency Domain & Hybrid Images
* **Frequency Filtering:** Utilizes the **Discrete Fourier Transform** to shift the image into the frequency domain, where **Low-Pass** (smoothing) and **High-Pass** (edge-preserving) masks are applied before inverse transformation.
* **Hybrid Images:** Generated by combining a low-pass filtered version of one image and a high-pass filtered version of a second image.
